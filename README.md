# arquitetura-limpa

Projeto do curso Arquitetura Limpa do FormacaoDEV

1. npm init -y
2. npm i -D typescript
3. npx tsc --init
4. criar .gitignore
5. criar pastas ./src e ./test
6. npm i -D jest ts-jest @types/jest @types/node
7. npm i -D ts-node-dev
8. npx ts-jest config:init
9. npm i bcrypt
10. npm i -D @types/bcrypt
11. npm i knex pg
12. npm install -D @types/pg
13. npm install dotenv
14. npm i -D ts-node typescript @types/node
15. npm i uuid
16. npm i -D @types/uuid
17. npm i express jsonwebtoken
18. npm i -D @types/express @types/jsonwebtoken axios
